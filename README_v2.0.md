# CUBE-to-XMP v2.0 🎨

一个现代化的 LUT 转换工具，支持拖拽文件和颜色预览。

## ✨ 新功能亮点

### 🖱️ 拖拽支持
直接将 `.cube` 或 `.xmp` 文件拖放到应用窗口即可转换，无需点击按钮。

### 👁️ 颜色预览
实时显示 LUT 对颜色的影响，帮助你快速评估转换效果。

### 📐 更大的窗口
默认窗口尺寸从 600x400 增大到 800x500，提供更好的操作体验。

## 🚀 快速开始

### 安装依赖
```bash
pip install customtkinter tkinterdnd2
```

### 运行应用
```bash
python cube_to_xmp.py
```

## 📚 功能特性

### 双向转换
- **CUBE → XMP**: 将 3D LUT 转换为 Adobe Camera Raw 配置文件
- **XMP → CUBE**: 将 Adobe 配置文件转换为标准 3D LUT 格式

### 智能处理
- 自动识别文件格式
- 大尺寸 LUT 自动插值（支持 64x64x64 及更大）
- 多线程异步处理，保持界面响应

### 预览功能
- 显示 6 个基本颜色的变换效果
- 左侧：原始颜色
- 右侧：应用 LUT 后的颜色

### 进度反馈
- 实时进度条显示
- 支持取消长时间操作
- 清晰的状态提示

## 🎯 使用场景

### 摄影师
将电影级的 `.cube` 调色预设导入 Lightroom Classic

### 调色师
在视频和照片后期之间共享调色预设

### 后期处理
在不同软件之间转换调色配置

## 🔧 技术规格

- **Python**: 3.11+
- **GUI 框架**: CustomTkinter
- **拖拽库**: tkinterdnd2
- **压缩算法**: zlib (Deflate)
- **编码格式**: Adobe Base85
- **LUT 格式**: 3D LUT (.cube)
- **配置格式**: Adobe XMP (.xmp)

## 📖 使用方法

### 方法 1: 拖放文件
1. 启动应用
2. 将 `.cube` 或 `.xmp` 文件拖放到窗口
3. 选择保存位置
4. 完成转换

### 方法 2: 使用按钮
1. 启动应用
2. 点击"选择文件并转换"
3. 选择要转换的文件
4. 选择保存位置
5. 完成转换

### 方法 3: 导出预设
1. 切换到"内置胶片预设"标签
2. 选择要导出的预设
3. 点击"导出所选预设"
4. 选择保存位置

## ⚙️ 配置选项

### 语言设置
- 中文
- English

### 主题设置
- System (跟随系统)
- Dark (深色模式)
- Light (浅色模式)

## 🐛 已知限制

1. XMP 最大支持 32x32x32 的 LUT 尺寸
2. 预览仅显示基本颜色，非真实图片效果
3. 需要安装 tkinterdnd2 库才能使用拖放功能
