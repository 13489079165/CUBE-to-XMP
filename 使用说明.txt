# 🚀 快速启动 CUBE-to-XMP v2.0

## 方式 1: 直接运行（推荐测试）

### 步骤：

1. **确保已安装 Python**
   - 下载 Python 3.10+：https://python.org
   - 安装时勾选 "Add Python to PATH"

2. **双击运行 `启动程序.bat`**
   - 脚本会自动检查并安装依赖
   - 自动启动程序

3. **开始使用！**
   - 拖放 .cube 或 .xmp 文件到窗口
   - 或点击"选择文件并转换"按钮

---

## 方式 2: 构建独立 exe（分发给其他人）

### 步骤：

1. **双击运行 `build_windows.bat`**
   - 脚本会自动安装 PyInstaller
   - 自动构建 Windows 可执行文件

2. **等待构建完成（约 2-5 分钟）**

3. **在 `dist/CUBE-to-XMP/` 中找到：**
   - `CUBE-to-XMP.exe` - 可执行文件
   - `_internal/` - 包含所有依赖

4. **压缩整个文件夹分享给用户**
   - 用户解压后双击 exe 即可使用
   - 无需安装 Python 或任何依赖

---

## 功能亮点

✅ **拖拽支持** - 直接拖放文件到窗口

- **Python**: 3.10+ (仅方式 1)
- **内存**: 4GB RAM
- **磁盘空间**: 100MB (方式 1), 20MB (方式 2)

## 常见问题

### Q: 运行 .bat 文件时提示"Python 不是内部命令"
A: 请重新安装 Python，并在安装时勾选 "Add Python to PATH"

### Q: 启动程序.bat 闪退
A: 右键点击文件 -> 选择"以管理员身份运行"

### Q: 如何切换语言？
A: 在程序右上角的下拉菜单中选择"中文"或"English"

### Q: 支持哪些文件格式？
A: 支持 .cube (3D LUT) 和 .xmp (Adobe Camera Raw 配置文件)
