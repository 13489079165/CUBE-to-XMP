# CUBE-to-XMP v2.0 - 新功能介绍

## 🚀 主要改进

### 1. 窗口大小优化
- **默认窗口尺寸**: 从 600x400 增大到 **800x500**
- **最小尺寸**: 从 500x350 增大到 **600x400**
- 提供更好的视觉体验和操作空间

### 2. 拖拽文件支持 ✨
现在可以直接拖放文件到应用窗口进行转换！

**使用方法:**
- 将 `.cube` 或 `.xmp` 文件直接拖放到应用窗口
- 程序会自动识别文件格式并开始转换
- 无需点击按钮，更直观的操作方式

**支持的文件格式:**
- `.cube` - 3D LUT 文件（自动转换为 .xmp）
- `.xmp` - Adobe Camera Raw 配置文件（自动转换为 .cube）

### 3. 颜色预览功能 🎨
实时预览 LUT 对颜色的影响！

**功能特点:**
- 显示 6 个基本颜色的预览（红、绿、蓝、黄、品红、青）
- 左侧显示原始颜色
- 右侧显示应用 LUT 后的颜色
- 直观对比转换前后的效果

**应用场景:**
- 快速评估 LUT 的效果
- 预览色彩变换
- 选择最合适的 LUT

## 🎯 使用指南

### 快速开始
1. **拖放文件** - 直接将文件拖到窗口
2. **选择文件** - 点击"选择文件并转换"按钮
3. **导出预设** - 从内置预设中导出

### 转换流程
1. 拖放或选择文件
2. 程序自动识别格式
3. 显示进度对话框（支持取消）
4. 实时显示颜色预览
5. 选择保存位置
6. 完成转换

## 🔄 转换对比

### CUBE → XMP
```
输入: .cube 文件
输出: .xmp 文件（Adobe Camera Raw 配置）
预览: 实时显示颜色变换
```

### XMP → CUBE
```
输入: .xmp 文件
输出: .cube 文件（标准 3D LUT 格式）
预览: 重置为原始颜色
```

## 🐛 已知问题

### 限制
- XMP 文件的最大 LUT 尺寸为 32x32x32
- 预览仅显示基本颜色，非真实图片
- 拖放功能需要 tkinterdnd2 库

### 解决方案
- 程序会自动处理大尺寸 LUT 的插值
- 颜色预览提供直观的参考
- 依赖库会自动安装
